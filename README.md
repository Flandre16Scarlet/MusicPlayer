# MusicPlayer
Windows上的音乐播放器，使用开源libzplay。


项目说明：

    12年大学时学习C++时练手，用MFC自绘界面+libzplay音频播放库开发的一个Windows的音乐播放器。
    主要包括：基础功能、界面自绘、歌词面板、截屏、MFC界面功能等模块。
    
    
编译：

    环境：VS2008
    步骤：打开MusicPlayer.sln项目文件编译即可。
